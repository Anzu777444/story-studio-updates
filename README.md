# Story Studio — userscript updates

This repo hosts the auto-updating **Story Studio** userscript for patrons.

## Install (one time)
1. Install the **Tampermonkey** browser extension (or Violentmonkey).
2. Click **[story-studio.user.js](story-studio.user.js)** above, then click the **Raw** button.
3. Tampermonkey opens an install page → click **Install**.

## Updating
Once installed, you're on the auto-update track:
* Tampermonkey checks for new versions on its own and will offer the update.
* When you open **Story Studio**, if a newer version exists you'll see a
  **"🔔 A new version is available"** banner — click **Update now** and confirm the install.

Current version: **v1.0.14**
